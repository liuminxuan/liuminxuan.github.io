N2日语学习笔记（三）

发表于 2019-01-25 | 更新于: 2019-02-04 | 分类于 N2日语学习笔记

～だけあって/だけに/だけのことはある

  • “不愧是…” =さすがに
  • 「だけに」“正因为…” =「〜だけに、かえって」”正因为…反而更加…”

たとえ〜ても

  • 表示让步，“即使…也”
  • 「たとえ」哪怕

～たところ

  • 动た形＋ところ
  • ＝〜たら
  • 「ところだ」
    • 「动原形+ところだ」”正要…”
    • 「动た形+ところだ」“刚刚…（非常刚刚）”
    • 「动ている形＋ところだ」 “一段时间内正在…”

～たところが

  • ＝たら　得到意料之外的结果（与预料中相反）
  • ＝が “虽然…但是…”

～たとたん(に)

  • 汉字：た途端(に)
  • 前项刚一发生就发生了后项意想不到的事情，句末只能用过去式
  • 后项不能是命令、意志、推量、否定的内容

～たばかり

  • 刚刚做完某事，隔了没多久
    • 如：火车站去年刚刚建好。
  • 「たところだ」非常刚刚的刚刚
    • 如：刚刚才吃完饭。

～たびに

  • 汉字：度に
  • 客观的每次每当，如：每次他出差就会给我带好吃的。
  • 「ずつ」每一次变化上的数量。
  • 「につけ」每当…都…（主观的），如：每当看到这张照片，都会想起他。
  • 「毎に/ごとに」每（10公里）
  • 「につき」平均每/（正式的）因为
  • 「おきに」每隔

～だらけ

  • 名　+ 〜
  • “净是…”，前项名词一般带负面含义

～ついでに

  • 动原形/た形/名の＋〜
  • ”顺便…”，做前项事情，顺便做后项事情（两项事情不是一件事）

～っこない

  • 多用于口语，“不可能…”

〜つつ/つつも

  • 动ます形＋〜
  • ”一边…一边…”/“虽然…但是…”
  • =ながら

〜つつある

  • 动ます形＋〜
  • 表示某一种延续性动作正在进行，或者某个变化在不断发展。

〜っぽい

  • 表示某种感觉或倾向。
  • 「子どもっぽい」孩子气的

〜つもりだ/つもり

  • 打算
  • 「た/ている形」自己认为，表达主语自己的想法。

〜つもりで

  • 动た形＋〜
  • “就当是…”，以前项的心态做后项
  • 「动た形＋ことにする」对事情的处理结果
    • 如：别人骂就当作听不见

〜て以来

  • “…以来”

〜てからでないと/てからでなければ

  • 表示不先做前项，就不能做后项。

〜てしょうがない/てしようがない

  • “…得受不了”，口语
  • 主语是第三人称时，要在句末加上「らしい/そうだ/ようだ」

〜てたまらない

  • 汉字：て堪らない
  • ＝てしょうがない/てしようがない

〜てでも

  • =でも

〜てならない

  • 情感上的不得了

～てはいられない

  • “不能”迫于某种情况不能以前的状态，包含紧迫感和危机感

〜てばかりはいられない/てばかりもいられない

  • “不能总是这么下去，不能沉浸在…中”

〜てはじめて

  • 「AてはじめてB」做了A之后再做B。

〜ということだ

  • 解释说明。
  • 听说传闻
    • 「そうだ」「らしい」「って」「という」「とか(非常不确定)」

〜というものではない/というものでもない

  • 「というものだ」的否定
  • “并非”

〜というより

  • “与其说…不如说…”。

〜といっても

  • “虽说 …”
  • 注意与「からといって」区别

〜とおり(に)/どおり(に)

  • 动原形/た形/名の+～
  • =と同じに
  • 名词直接加「どおり」

〜とか

  • 表示传达的内容不确定。
  • 「とかいうことだ」听说（不确定的）。
  • 「とかで」（不确定的）因为。
