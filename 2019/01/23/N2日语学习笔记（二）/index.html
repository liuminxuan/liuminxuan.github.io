N2日语学习笔记（二）

发表于 2019-01-23 | 更新于: 2019-01-25

～からいうと/からいえば/からいったら/からいって

  • “从…来说”

～からこそ

  • “正因为…”

～からして

  • 单从…来判断

～からすると/からすれば

  • “从…来判断”

～からといって

  • 虽说，承认前项但是不承认前项可以推出后项，“不能因为…就…”
    • 虽说工作忙，但不能因此就不吃早饭
  • 「といっても」虽说，后项不是典型的前项描述事物，对前项的补充说明。
    • 虽说找到的工作，但是时给并不高。

～からには

  • 三个“既然”中的一个

～から見ると/から見れば/から見て/から見ても

  • “从…来看”

～代わりに

  • 代替，交换
  • 同一事物的正反两面
    • 这份工作虽然稳定，但是工资没有那么高

～気味

  • 动ます形 + ～
  • 「〜がちだ」“容易…”

～きり/きりだ

  • 表示限定=だけ
  • “一…就再没…”
    • 她离家出走后，就再也没回来。
  • 「动ます形＋〜」维持状态

〜きる/きれる/きれない

  • 表示程度深，「思い切る」下决心
  • 表示做（不）完某个动作
    • 天上的星星数不清
  • 「かけて」“没做完…”

〜くせに/くせして

  • =のに
  • 带有指责、抱怨、不满的语气

〜くらい/ぐらい

  • 表示程度
  • 表示事物的最高程度“没有比…更…”
    • =ほど
    • 没有比熬夜更有损健康的了。

〜ことか

  • 表达非常强烈的感情，“多么…啊！”
  • 如：「何度注意したことか」我提醒你多少次了啊。

〜ことから

  • 表示原因，“因为…”
  • 「～のことだから」通常接人，表示该人物的典型特征。

〜ことだ

  • 表示忠告和建议
  • 去掉だ，表示规章制度
  • 「わけだ」解释原因

〜ことだから

  • 通常接人，表示该人物的典型特征。
  • 如「劉さんのことだから」因为是小刘嘛

〜ことなくこともなく

  • 书面用语
  • = ずに/ないで

〜ことは〜が

  • 退让性转折
  • 如：「読んだことは読んだが、内容が全然分からない」读是读了，内容完全不懂。

〜ことはない

  • =「必要はない」
  • 「はずがない」“不应该”
  • 「わけがない」“不可能”
  • 「わけではない」“并不是…一回事”

〜際(は)/際に(は)

  • =〜節に　在…节点上

～最中(に)/最中だ

  • 表示正在做某事时，突然发生后项的事情

〜ざるを得ない

  • 迫于外界压力，不得不做某事
  • 「ずにはいられない」自己主动的不得不

〜次第/次第だ/次第で

  • 次第在N2语法中一共3个意思熟记
    • “一…就…” =途端
    • 解释说明 =わけだ
    • 取决于 =によって

〜上/上は/上も

  • “从…来看”
  • 常接在「法律、教育、経済、理論、書類」等词汇后面
  • 「上に」而且
  • 「上は」既然

〜末(に)/末の

  • 经过一番讨论后得到的结果。

〜ずにすむ/ないですむなくてすむ

  • 汉字：済む
  • するー＞（せ）ずにすむ

〜ずにはいられない/ないではいられない

  • 表示控制不住做某事的心情，如果不做某事心里会不舒服，情不自禁。
  • 「ざるを得ない」迫于外界原因不得不做某事。

〜せいで/せいか/せいだ/せいにする

  • “都怪…”
  • 将过错推到前项

〜(たい)ばかりに/(ほしい)ばかりに

  • “只是因为…”
  • 「だけに」“正因为…”
  • 「からこそ」“正因为…”
