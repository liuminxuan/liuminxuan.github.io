N2日语学习笔记（一）

发表于 2019-01-23 | 更新于: 2019-01-28 | 分类于 N2日语学习笔记

～あげく

  • 汉字：挙句
  • 接续：动词た形/名词の　＋　あげく(に)
  • 表示不好的结果，“最后…”
  • 易混淆：「〜末に」表示经过一番讨论后的结果。

～あまり(に)

  • 汉字：～余り(に)
  • ”由于过度…而…”相当于英文的“too…to…”
  • 常与表示状态与感情的词连用。

～以上(は)

  • “既然…就…”
  • N2语法中只有三个词有既然的意思，分别为 “上は、以上は、からには“

～一方/一方で(は)

  • “一方面…另一方面…”
  • 可以用于顺承，也可用于转折。

～一方だ

  • “不断…”,“越来越…”表示某种倾向，状态朝一个方向不断发展。
  • 易混淆：「〜ばかりだ」表示消极的持续动作（不能接名词）。比如光知道玩“遊ぶばかりだ”

～上で(は)

  1. 动词原形/名の＋上で(は)
    • “在…方面”
    • 名词情况下，常与「日历」「法律」「地图」「数据」等连接。
  2. 动词た形/名の＋上で(は)
    • “在…之后”
    • 有时用「名+の＋上」，用于通知书等书面语。

～上は

  • “既然…就…”

～うちに

  1. “在…期间”,”趁着…”
  2. 在某事发生期间，发生了另外一件事
    • 「新买的电脑还没过一个月就坏了」
    • 前项表示持续性的动作，后项表示没有预料到并且是不知不觉的没有预料的结果。

～うちは

  • “在…期间””趁着…”
  • 可用于别人的事情，也可用于自己的事情。

～(よ)うではないか/(よ)うじゃないか

  • 表示建议或劝诱（不是反问）。

～得る

  • 动ます形+得る
  • ＝动词的可能形（表示可能性，不是能力）
  • 常用形：ありうる

～得ない

  • 动ます形+得ない
  • 没有这种客观上发生的可能性
  • 「(せ)ざるを得ない」不得不
  • 常用形：ありえない

～おかげで/おかげだ

  • 汉字：お陰で/お陰だ
  • “托…的福”“多亏了…”，表示原因
  • 常用于好的结果，有时也有反话
  • 「～せいだ」“都怪…”

～恐れがある

  • 有可能发生不好的结果
  • ＝「〜かねない」客观上的有可能

～甲斐がある/甲斐がない/甲斐もなく

  • 动た形/名の＋　〜
  • “(没)有…的价值”

～がきっかけで/がきっかけになって/をきっかけに(して)/をきっかけとして

  • 「〜がきっかけになって」自己成为契机　　　
  • 「をきっかけに(して)」 使…成为契机
  • =「～が契機で/が契機になって/を契機に(して)/を契機にして」

～限り(は)/限りでは/ない限り

  • 「限り」在n2中只有三个意思熟记
    • 尽量，尽最大程度：できる限り＝できるだけ
    • 只要…就…：只要他不跟我道歉，我就不原谅他
    • 据…所知,据…所调查：調べる限り

～が契機で/が契機になって/を契機に(して)/を契機にして

  • 书面语
  • =「～がきっかけで/がきっかけになって/をきっかけに(して)/をきっかけとして」

～かけだ/かけの/かける

  • 动ます形 + ～
  • “…做到一半”
  • 「死にかけていた猫」差点死掉的猫。

～がたい

  • 表示主观上的“难以”，「难以忘却」「难以置信」「难以原谅」
  • 「動かしがたい事実」难以否认的事实
  • 「かねる」客观的难以，如“经理不在，我难以回答您的问题。”

～がちだ/がちの/がちな

  • 表示负面的倾向，容易发生某种负面的消极的事情，“容易…”“往往会…”
  • 「病気がちだ」容易生病

～(か)というと/(か)といえば

  • “要说是否…”

～(か)と思うと/(か)と思ったら

  • かと思うと：刚…就…/原以为…却…/一想到…就…
  • かと思えば：原以为…却…/既…又…
  • かと思ったら：刚…就…/原以为…却…

～かと思えば

  • 同上
  • “既…又…”常与「ある/いる」搭配

～か〜ないかのうちに

  • 「动原形＋〜」“还没…”
  • 「动た形＋〜」”已经（刚刚）”

〜か何か

  • 表示泛指。
  • 「何て」轻蔑的语气。

～かねない

  • 动ます形＋〜
  • 有可能有不好的结果
  • = 恐れがある

～かねる

  • 动ます形＋〜
  • ”（客观上的）难以”，即使想做到也很难做到。
  • 如“经理不在，我难以回答您的问题。”

～かのようだ/かのような/かのように

  • 动普通形 + ～
  • =「ようだ」

～からいいようなものの

  • 幸好
